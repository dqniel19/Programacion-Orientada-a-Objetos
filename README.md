## **Descripción**  
- La Programación Orientada a Objetos (POO) organiza el software en objetos con atributos y métodos, facilitando la reutilización y el mantenimiento.
Se basa en cuatro pilares: encapsulamiento, abstracción, herencia y polimorfismo, que permiten crear sistemas modulares y flexibles.
Estos conceptos se aplican según el contexto y serán explorados más a fondo en una exposición de Canvas.
